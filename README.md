# 🌟 Site Interativo com Animações GSAP, JavaScript e CSS

Bem-vindo ao meu projeto! Este é um site moderno e responsivo que combina **design impactante**, **animações fluidas com GSAP**, e **lógica interativa com JavaScript**. Feito para impressionar visualmente e oferecer uma experiência de navegação dinâmica e envolvente. 🚀

## 🧠 Sobre o Projeto

Este projeto foi criado com o objetivo de explorar o poder da **animação web**, unindo:
- 🎨 Estilização com **CSS puro**, com atenção a responsividade e estética clean.
- ✨ **GSAP (GreenSock Animation Platform)** para animações suaves e criativas.
- ⚙️ **JavaScript Vanilla** para adicionar interatividade e controle dinâmico dos elementos da página.

## 🔥 Funcionalidades

✅ Animações de entrada fluídas e sincronizadas  
✅ Interações ao rolar a página e ao passar o mouse  
✅ Layout 100% responsivo para todos os dispositivos  
✅ Código limpo e organizado para fácil manutenção

## 📸 Prévia

*A navegação ganha vida com cada interação!*

## 🛠️ Tecnologias Usadas

- HTML5
- CSS3
- JavaScript ES6+
- [GSAP](https://greensock.com/gsap/)
